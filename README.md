## Cách dùng
### Đối với file Jupyter Notebook
- Tải VS Code.
- Tải các extension cho Jupyter Notebook.
- Tải thư viện phù hợp:
``bash
!pip install <tên_thư_viện>
``
- Chạy file ipynb mong muốn.
### Đối với file R

- Tải thư viện phù hợp:
``bash
!pip install <tên_thư_viện>
``
- Chọn file R mong muốn.
